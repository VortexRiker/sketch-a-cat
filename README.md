# sketch-a-cat
This is a "sketchpad" project. It provides an ability for it's users to draw on a canvas by holding LMB and moving the cursor around the canvas area. User has an ability to change the size of the canvas grid cells, thus effectively changing the size of the brush they use.

Project means to test the application of learned Javascript topics like Events, and DOM manipulation, and also acts as a refresher on CSS FlexBox.
